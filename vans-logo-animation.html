Vans Logo Animation

Project Scope

The assignment for this project was to create an animation video for an existing logo. I chose to create my based on the Vans logo because it is a brand I love and know well. The goal was to create a logo that fits with the grunge, skater vibe the brand owns.

Design Process

I started this process by creating some rough sketches which describe my vision for the video. I did this is a storyboard style, roughing out various stages in the video to demonstrate the stages of the animation. After this process was complete my next step was to create a style frame for the video which included all the assets I would need.

Once all the assets were created and I had a plan for my video it was time to start creating.

As the main aspect of the video would be done through rotoscoping, I had to find some reference videos which I could mash together into one smooth sequence. Once I had found what I needed I began the process of rotoscoping.

The last step was to add some effects to the rest of the video and a mask reveal to the paint strokes to give it a more dynamic look. The final result suites the brand and gives a very street/skater feel.
